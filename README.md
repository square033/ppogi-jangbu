# 🧸 뽑기 장부

내가 뽑은 인형들을 사진과 함께 기록하고, 크기·카테고리·가격으로 정리해서 관리하는 나만의 인형 장부예요.

---

## 시작하기

### 처음 오셨다면 — 회원가입
1. 로그인 화면에서 **"회원가입"**을 눌러요.
2. 원하는 **아이디**(영문·숫자 3자 이상)와 **비밀번호**(6자 이상)를 입력해요.
3. **"회원가입"** 버튼을 누르면 바로 로그인돼요.

### 이미 계정이 있다면 — 로그인
아이디와 비밀번호를 입력하고 **"로그인"**을 누르면 돼요. 한 번 로그인하면 어떤 기기에서 접속해도 같은 인형 목록을 볼 수 있어요.

### 로그인 없이 구경만 하고 싶다면
로그인 화면 맨 아래 **"🧸 주인장 인형 구경하기"** 버튼을 누르면, 계정 없이도 주인장의 인형 컬렉션을 둘러볼 수 있어요. 단, 이 모드에서는 **보기만 가능**하고 추가·수정·삭제는 할 수 없어요. 화면 위쪽에 "구경 모드" 안내가 떠 있고, 다시 로그인 화면으로 돌아가려면 오른쪽 위 **←** 버튼을 누르면 돼요.

---

## 인형 등록하기

1. 화면 아래 **"＋ 인형 추가하기"** 버튼을 눌러요.
2. **사진 추가** 박스를 눌러 사진을 골라요.
3. 편집 화면이 뜨면:
   - 손가락으로 **드래그**해서 원하는 부분이 가운데 오도록 위치를 잡고
   - 아래 슬라이더로 **확대/축소**하고
   - **"↻ 90° 회전"**으로 사진을 돌린 다음
   - **"적용"**을 눌러요. 정사각형으로 딱 맞게 잘려서 저장돼요.
4. **이름**을 입력해요 (예: 산리오 시나모롤 특대).
5. **크기**(소형/중형/대형)를 골라요.
6. **카테고리**는 선택 사항이에요. 기존에 써둔 카테고리는 칩을 눌러 바로 고를 수 있고, 새 카테고리는 직접 입력하면 돼요.
7. 얼마쯤 나갈지 궁금하면 **"번개장터 시세" / "당근 시세"** 버튼으로 실제 거래가를 확인해볼 수 있어요.
8. **책정 가격**을 입력해요 (아직 안 정했으면 비워둬도 돼요).
9. 지금 판매 중인 글이 있다면 **판매 링크**도 넣을 수 있어요.
10. **"저장하기"**를 누르면 완료!

---

## 인형 목록 보기 · 관리하기

### 인형 카드 눌러보기
목록에서 인형 카드를 누르면 사진, 이름, 가격, 등록일 등 상세 정보를 볼 수 있어요. 여기서 **"수정하기"**를 눌러 정보를 바꾸거나, 편집 화면의 🗑 버튼으로 삭제할 수 있어요.

### 검색·필터·정렬
- **검색창**에 이름 일부만 입력해도 실시간으로 걸러져요.
- 상단 탭에서 **전체 / 소형 / 중형 / 대형**으로 크기별로 볼 수 있어요.
- 카테고리를 하나라도 등록했다면 그 아래 **카테고리 탭**이 자동으로 생겨요.
- 오른쪽 위 드롭다운으로 **가격대**(~5,000원, 5,000~1만원 등)를 필터링할 수 있어요.
- 왼쪽 드롭다운으로 **정렬 기준**(최신순·오래된순·가격순·이름순)을 바꿀 수 있어요.
- 이 필터들은 동시에 겹쳐서 적용돼요. 예: "산리오 + 소형 + 1만원 이하 + 가격 낮은순"처럼요.

### 한눈에 보는 요약
화면 위쪽 카드에서 **전체 책정 가치, 전체 마릿수, 가격 미정 개수, 평균가**를 바로 확인할 수 있어요.

## 로그아웃

화면 위쪽 오른쪽의 **⏻ 버튼**을 누르면 로그아웃돼요.

## 자주 묻는 질문

**Q. 다른 사람이 내 인형 목록을 볼 수 있나요?**
로그인한 계정의 인형은 본인만 볼 수 있어요. 다만 "주인장 인형 구경하기" 기능이 켜져 있다면, 주인장이 정한 계정 한 곳의 인형만 누구나 읽기 전용으로 볼 수 있어요.

**Q. 사진을 다시 자르고 싶어요.**
인형 상세 화면 → **수정하기** → 사진을 다시 눌러서 편집하면 돼요.

**Q. 가격을 아직 못 정했어요.**
책정 가격 칸을 비워두면 "가격 미정"으로 표시돼요. 나중에 언제든 수정에서 채워 넣을 수 있어요.
