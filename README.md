# 🧸 뽑기 장부

인형뽑기로 모은 인형들을 사진·이름·가격으로 정리하는 개인용 웹앱이에요.
토스 스타일 UI에 프리텐다드 폰트로 만들었고, 로그인하면 어느 기기에서든 같은 목록을 볼 수 있어요.

## 🔗 접속하기

👉 **https://square033.github.io/ppogi-jangbu/**

위 주소로 들어가면 바로 사용할 수 있어요. 폰 홈 화면에 추가하면 앱처럼 쓸 수 있어요.

## 📖 사용법

1. **로그인 / 회원가입**
   처음이면 "회원가입"을 눌러 원하는 아이디(영문·숫자)와 비밀번호를 만들어요.
   내 데이터는 나만 볼 수 있고, 다른 사람은 로그인해도 자기 것만 보여요.

2. **인형 추가**
   아래 `＋ 인형 추가하기` 버튼을 눌러요.
   - 📷 사진 추가
   - 이름 입력
   - 크기 선택 (소형 / 중형 / 대형)
   - 가격 입력 (나중에 정해도 돼요)

3. **시세 확인**
   이름을 입력하면 `번개장터 시세` · `당근 시세` 버튼이 활성화돼요.
   버튼을 누르면 그 이름으로 검색한 결과가 새 탭에서 열려요.
   실제 거래가를 보고 가격을 정한 뒤 입력하면 돼요.

4. **정리해서 보기**
   - 상단 카드에서 전체 마릿수와 책정 가치 합계를 한눈에 확인
   - 소형 / 중형 / 대형 탭으로 크기별 필터링
   - 인형 카드를 누르면 상세 정보 확인 및 수정·삭제 가능

## 🛠 기술 스택

- 순수 HTML / CSS / JavaScript (단일 파일)
- [Supabase](https://supabase.com) — 로그인 & 데이터 저장 (RLS로 개인 데이터 보호)
- [Pretendard](https://github.com/orioncactus/pretendard) 폰트
- GitHub Pages 배포

## 💡 참고

- 로그인은 아이디 기반이라 **비밀번호 재설정이 불가능해요.** 비밀번호는 꼭 기억해주세요.
- 데이터는 Supabase 서버에 저장되므로, 로그인만 하면 폰·PC 어디서든 동일하게 보여요.
